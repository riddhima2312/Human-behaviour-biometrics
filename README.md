# Human Behaviour Biometrics Dashboard

A modern AI-powered behavioral biometrics web application built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Recharts**.

## Features

- Real-time keystroke dynamics capture
- Dwell time, flight time, typing speed, and rhythm analysis
- Authentication confidence and anomaly detection indicators
- Interactive dashboard cards and line charts
- Responsive layout with reusable UI components
- Dark mode toggle and smooth UI animations

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts

## Project Structure

- `src/app` - app shell and page
- `src/components` - reusable UI and dashboard components
- `src/components/charts` - chart components
- `src/lib` - biometrics analysis logic
- `src/types` - shared TypeScript types

```bash
npm run dev
```
